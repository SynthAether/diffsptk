diffsptk
========
*diffsptk* is a differentiable version of SPTK based on the PyTorch framework.

[![](http://img.shields.io/badge/license-Apache%202.0-green.svg)](https://github.com/sp-nitech/diffsptk/blob/master/LICENSE)


Requirements
------------
- PyTorch 1.8.0+
